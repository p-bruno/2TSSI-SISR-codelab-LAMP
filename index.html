Importation du fichier 2TSSI-SISR-BRUNO.ova

Ouvrir l'explorateur de fichier

Si vous ne possédez pas le fichier, alors connectez-vous au NAS puis copier/coller le fichier /bruno/2TSSI/VM multi-usages/2TSSI-SISR-BRUNO.ova dans le dossier /Bureau/2TSSI/VM multi-usages/2TSSI-SISR-BRUNO.ova

Le dossier /BRUNO du serveur NAS est un dossier distant.

Durant l'année nous allons essentiellement travailler avec des fichiers en local. Pour cela l'enseignant placera les fichiers :

  • Soit sur le serveur NAS : il faudra alors les copier/coller sur votre clef USB
  • Soit sur le dossier /partage/bruno lorsque les fichiers sont volumineux. De même, il faudra copier/coller ces fichiers sur votre clef USB.
